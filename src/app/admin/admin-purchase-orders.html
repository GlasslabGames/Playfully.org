Purchase Orders

Use this page to handle all purchase order transactions. Here you can indicate you've received a purchase order (which will grant the owner premium access), approve a purchase order, or reject a purchase order. Each action requires the same information in order to validate:

  • key: a unique identifier associated with the form, created by GlassLab, Inc.
  • PO#: the number on the form
  • payment: the amount being paid to GlassLab, Inc.
  • plan: the package plan type
  • seats: the package seats size
  • action: received, approved, rejected
