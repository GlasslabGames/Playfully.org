SimCityEDU Technical Requirements

To ensure a smooth SimCityEDU experience, this page tests your environment for the minimum requirements needed to run the SimCityEDU dashboard application.

Tests verify the below requirements:

Browser
  • Chrome 29+
  • Firefox 26+
Operating System
  • Windows 7+
  • Mac OSX 10.7.5+
Cookies
  • Cookies enabled.
Screen Size
  • 1024 x 500 Minimum.
Server Connection
  • Valid connection to server.
Connection Speed
  • 750 kbps Minimum.
